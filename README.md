AgriGenAI: The Crop Saver 

Multi-Agent System using Langflow & IBM Watsonx.ai 
Project on Agriculture domain: " AgriGenAI: The Crop Saver" (Multi-Agent solution) 
The Challenge: - 
Farmers often spot symptoms on their crops (e.g., "yellow leaves," "black spots") but struggle to distinguish between fungal infections, nutrient deficiencies, or pests. Misdiagnosing leads to spraying the wrong chemicals, wasting money and harming the soil. 
The Objective 
Build a Multi-Agent AI system that helps farmers accurately diagnose crop issues based on symptoms and provides actionable treatment recommendations. The system reduces misdiagnosis, prevents unnecessary chemical usage, and promotes sustainable farming practices.
